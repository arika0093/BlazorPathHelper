    Title: Feature List¶

    Description: This document explains the main features offered by this library.¶

    This library automatically generates the following:

    • A type-safe URL Builder from URL definitions
      • Supports parameters, optional parameters, and queries
    • Automatic definition of routes, parameters, and queries
    • Automatic generation of menu structure data
      • Supports customization of menu display (icons, descriptions, localization, etc.)

    These features are independent of each other and can be used partially as needed. For more details on each feature, please refer to the links above. To get an overview of the entire library, check out Getting Started.
